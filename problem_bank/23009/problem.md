# 23009  [VH2505]. 연속하는 겹치는 구간의 수 (Number of Consecutive Overlapping Intervals)

- **Difficulty**: D7 (VH)
- **Point**: 200
- **Accuracy**: 6.67%

## Constraints

- **Time**: 12 test cases combined — C: 20s / C++: 20s / Java: 30s / Python: 30s
- **Memory**: Heap + static memory ≤ 256MB, Stack ≤ 1MB

## Problem

1부터 N까지의 자연수가 한 번씩 나타나는 길이 N의 순열 p₁, p₂, ···, pN이 있다.

이 문제에서 "구간 [l,r]"은 l 이상 r 이하의 모든 자연수로 구성된 집합을 의미하며, l, r이 자연수라는 것을 암시한다.

어떤 구간 [l,r]이 **연속한다**는 것은, 순열의 해당 구간 p_l, p_{l+1}, ···, p_r에 있는 r – l + 1개의 자연수들이 연속한 구간을 이룬다는 것이다.

즉, max(p_l, p_{l+1}, ···, p_r) – min(p_l, p_{l+1}, ···, p_r) = r - l 이 성립한다는 것이다.

두 구간 [l₁,r₁], [l₂,r₂]가 **포함 관계**라는 것은, l₁ ≤ l₂ ≤ r₂ ≤ r₁ 또는 l₂ ≤ l₁ ≤ r₁ ≤ r₂가 성립한다는 것이다.

두 구간 [l₁,r₁], [l₂,r₂]가 **겹친다**는 것은, 두 구간이 포함 관계가 아니면서 두 구간에 동시에 속하는 자연수가 존재한다는 것이다.

아래 조건을 모두 만족하는 네 자연수의 쌍 (l₁, r₁, l₂, r₂)의 개수를 998,244,353으로 나눈 나머지를 구하는 프로그램을 작성하라.

1. [l₁,r₁]과 [l₂,r₂]는 **연속하는** 구간이다.
2. [l₁,r₁]과 [l₂,r₂]는 서로 **겹친다**.
3. [l₁,r₁]과 [l₂,r₂]의 교집합에는 X개 이상의 자연수가 들어 있다.

## Input

첫 번째 줄에 테스트 케이스의 수 T가 주어진다.

각 테스트 케이스는 두 개의 줄로 이루어진다.

첫 번째 줄에는 두 개의 정수 N(1 ≤ N ≤ 300,000)과 X(1 ≤ X ≤ N)가 공백 하나를 사이로 두고 주어진다.

두 번째 줄에는 p₁, p₂, ···, pN이 공백 하나를 사이로 두고 주어진다.

## Output

각 테스트 케이스에 대해, 조건을 만족하는 네 자연수 순서쌍의 개수를 998,244,353으로 나눈 나머지를 출력한다.

## Sample

### Input
```
3
7 3
1 2 3 4 5 6 7
6 2
1 3 5 2 4 6
8 2
8 7 6 5 1 2 3 4
```

### Output
```
30
2
12
```
