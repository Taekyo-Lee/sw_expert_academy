# 24702. 2의 거듭제곱

- **Difficulty**: D7 (VH)
- **Point**: 200
- **Accuracy**: 30.00%

## Constraints

- **Time**: 12190 test cases combined — C: 10s / C++: 10s / Java: 20s / Python: 30s
- **Memory**: Heap + static memory ≤ 256MB, Stack ≤ 1MB

## Problem

자연수 N에 대해, 2^N의 마지막 100자리 (단, 2^N이 100자리 이하면 2^N 그 자체)를 f(N)이라고 하자.

예를 들어 f(31) = "2147483648", f(12345) = "0544029064930412569943169729238102162312218687930203068055400275795180972382856696655279408212344832"이다.

1 이상 10^11 이하의 자연수 X가 주어진다.

X가 f(N)에 부분문자열로 나타나도록 하는 1 이상 10^50 미만의 자연수 N을 구하는 프로그램을 작성하라.

## Input

첫 번째 줄에 테스트 케이스의 수 T가 주어진다. 채점 시 사용되는 테스트 케이스는 1만 개 정도이다.

각 테스트 케이스는 한 개의 줄로 구성되며, 각 줄에는 1 이상 10^11 이하의 자연수 X가 주어진다.

## Output

각 테스트 케이스에 대해, X가 f(N)에 부분문자열로 나타나도록 하는 1 이상 10^50 미만의 자연수 N을 아무거나 하나 출력한다.

이러한 N이 반드시 존재함을 증명할 수 있다.

## Note

- 두 번째 예제: f(20) = "10**485**76"
- 세 번째 예제: f(64611113334280830274326754185942310046482626355574) = "76418060949607875427845202740795164936070957710**123456**35271811486670267403830961512317881157664374784"

## Sample

### Input
```
3
1024
485
123456
```

### Output
```
10
20
64611113334280830274326754185942310046482626355574
```
