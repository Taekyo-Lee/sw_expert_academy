## Problem Description

1번부터 N번까지의 번호가 붙은 N종류의 공이 있다.

i(1 <= i <= N)번 종류의 공은 F_i (1 <= F_i <= 2)개 있고, W_i (0 < W_i <= 1)의 실수 가중치가 붙어 있다.

F_1 + F_2 + ... + F_N = 30 (즉, 공의 총 개수는 30개)이다.

소정이는 다음과 같은 작업을 정확히 Q번 반복하였다.

1. 상자에 위에서 명시된 30개의 공을 넣는다.
2. 아래와 같은 방식으로 상자에서 공을 30번 뽑고, 뽑힌 공들의 종류를 순서대로 기록한다.
   - 현재 상자에 i번 종류의 공이 c_i개 들어 있다고 하자.
   - 모든 i (1 <= i <= N)에 대해, i번 종류의 공이 뽑힐 확률은 **(c_i * W_i) / (sum of c_j * W_j for all j)** 이다.
   - 뽑은 공은 작업이 끝나기 전까지 다시 상자에 넣지 않는다.

소정이가 공을 뽑은 기록이 모두 주어질 때, 모든 공 종류의 가중치를 역산하는 프로그램을 작성하라. 채점 방식은 "출력"을 참고하라.

## Input

첫 번째 줄에 테스트 케이스의 수 T가 주어진다.

각 테스트 케이스의 첫 번째 줄에는 Q(Q = 30000)와 N(1 <= N <= 30)이 주어진다.

두 번째 줄에는 N개의 정수 F_1, F_2, ..., F_N (1 <= F_i <= 2, sum(F_i) = 30)이 공백 하나씩을 사이로 두고 주어진다.

다음 Q개의 줄에는 sum(F_i) (= 30)개의 정수가 공백 하나씩을 사이로 두고 주어지는데, 이는 소정이가 한 공 뽑기 작업에서 뽑힌 공들의 종류를 순서대로 나열한 것이다.

각 줄에는 F_i개의 i가 주어진다. 소정이는 실제로 문제 본문에 기술한 방법으로 공을 뽑았으며, 이 점을 활용하여 문제를 해결해야 함에 유의하라.

## Output

각 테스트 케이스에 대해 N개의 실수 C_1, C_2, ..., C_N (10^{-300} <= C_i <= 1)을 출력한다.

모든 1 <= i, j <= N에 대해, W_i <= W_j 이면 |C_i/C_j - W_i/W_j| < 0.02 가 성립하면 정답 처리된다.

채점 프로그램은 무조건 N개의 실수를 출력하므로, 해결할 수 없는 입력이 주어질 경우 N개의 실수를 아무렇게나 출력하는 것을 권장한다.

**참고:** 아래 입출력 예제는 Q != 30000이고 sum(F_i) != 30이므로 입력 조건을 만족하는 예제가 아니다. 입력 형식을 보여주기 위해 임의로 만든 예시이다.

## Sample

### Input
```
1
2 4
2 2 2 1
3 4 3 1 1 2 2
3 3 4 1 1 2 2
```

### Output
```
0.1 1e-3 1 0.5
```
