# 24529. 격자판 위의 달팽이

- **Difficulty**: D6 (VH)
- **Point**: 150
- **Accuracy**: 7.67%

## Constraints

- **Time**: 47 test cases combined — C: 10s / C++: 10s / Java: 15s / Python: 25s
- **Memory**: Heap + static memory ≤ 256MB, Stack ≤ 1MB

## Problem

2×N 크기의 격자판과, 길이 2N의 정수 수열 A₁, A₂, ···, A₂ₙ이 있다.

당신은 격자판의 각 칸에 수열 A의 원소를 하나씩 적어야 한다.

격자판이 모두 채워지고 나면, 달팽이 한 마리가 격자판의 1행 1열에서 2행 N열까지, 오른쪽 또는 아래 방향으로만 이동할 것이다.

달팽이는 이동 과정에서 정확히 N+1개의 칸을 방문하며, 자신이 방문한 칸들에 적혀 있는 수들의 합을 최대화하는 경로를 택하여 방문한다.

당신은 달팽이가 지나가는 경로에 있는 수들의 합을 최소화하도록 격자판을 구성하고자 한다.

가능한 방법들 중 하나를 구하여 출력하는 프로그램을 작성하라.

## Input

첫 번째 줄에 테스트 케이스의 수 T가 주어진다. 각 테스트 케이스는 아래와 같이 구성된다.

첫 번째 줄에는 N (2 ≤ N ≤ 25)이 주어진다.

그 다음 줄에는 2N개의 정수 A₁, A₂, ···, A₂ₙ (0 ≤ Aᵢ ≤ 50000)이 공백 하나씩을 사이로 두고 주어진다.

## Output

각 테스트 케이스에 대해, 2개의 줄을 출력한다.

각 줄에는 공백 하나씩으로 구분된 N개의 정수를 출력해야 한다.

당신이 출력한 2N개의 정수는, 주어진 수열 A₁, A₂, ···, A₂ₙ과 동일 (정확히는, 모든 정수의 등장 횟수가 동일)해야 한다.

또한, 달팽이가 지나가는 경로에 있는 수들의 합이 최적의 방법과 동일해야 한다.

## Sample

### Input
```
2
2
1 2 3 4
3
1 1 1 1 2 2
```

### Output
```
1 4
3 2
1 1 2
1 2 1
```

## Note

첫 번째 예제: (1, 1)에서 (2, 2)로 가는 모든 가능한 경로는 (1, 1) → (1, 2) → (2, 2) 또는 (1, 1) → (2, 1) → (2, 2)이다.

각각의 합은 1+4+2 = 7, 1+3+2 = 6이다. 달팽이는 합이 더 큰 경로 (1, 1) → (1, 2) → (2, 2) 경로를 택하여 이동할 것이다.

격자판을 어떤 식으로 구성하더라도 달팽이가 지나는 경로의 길이는 7 이상이다.
